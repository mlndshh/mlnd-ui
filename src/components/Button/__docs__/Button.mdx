# Button

Button component with different props.

#### Example

<Canvas of={Button.PrimaryMedium} />

## Usage

```tsx
import { Button } from "sld-ui";

const Example = () => {
  return (
    <Button
      size="medium"
      text="Button"
      onClick={() => console.log("Clicked")}
      primary={true}
    />
  );
};

export default Example;
```

#### Arguments

- **text** `string` - A string that represents the text content of the button.
- **primary** `boolean` - A boolean indicating whether the button should have a primary styling or not. Typically, a primary button stands out as the main action in a user interface.
- **disabled** `boolean` - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
- **size** `"small" | "medium" | "large"` - A string that defines the size or dimensions of the button.
- **onClick** `React.MouseEventHandler<HTMLButtonElement>` - A function that is called when the button is clicked.
