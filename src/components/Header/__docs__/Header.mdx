# Header

A flexible heading component that supports all 6 HTML heading levels (h1-h6).

#### Example

<Canvas of={Header.Heading1} />

## Usage

```tsx
import { Header } from "sld-ui";

const Example = () => {
  return (
    <>
      <Header level={1}>Main Heading</Header>
      <Header level={2}>Subheading</Header>
      <Header level={3} className="text-blue-600">
        Custom Styled Heading
      </Header>
    </>
  );
};

export default Example;
```

## Custom Styling

You can add custom classes to modify the appearance:

<Canvas of={Header.CustomStyled} />

#### Arguments

- **level** `1 | 2 | 3 | 4 | 5 | 6` - Determines which heading element to render (h1-h6).
- **children** `ReactNode` - The content of the header.
- **className** `string` (optional) - Additional CSS classes to apply to the header.
